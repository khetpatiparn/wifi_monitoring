# wifi_strength_monitoring

### Step1 : create evn ###
linux
```
python3 -m venv .venv
```
windows
```
py -3 -m venv .venv
```
### Step2 : activate env ###
linux
```
. .venv/bin/activate
```
windows
```
.venv\Scripts\activate
```
### Step3 : install dependencies ###
```
pip install flask
```
```
pip install flask-socketio 
```
